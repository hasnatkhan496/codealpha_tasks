# codealpha_tasks
A console-based CGPA Calculator built in C++ using Object-Oriented Programming.
Supports multiple semesters and courses per semester. Each course stores its name,
grade (0.0–4.0 scale), and credit hours. The program computes per-semester GPA and
the overall Cumulative GPA (CGPA), then displays a formatted academic report.

Key Concepts: OOP (Classes, Encapsulation), Arrays, Const Methods, Formatted I/O
