# codealpha_tasks
# OOP Mini Projects — C++

Three beginner-to-intermediate C++ projects demonstrating core Object-Oriented
Programming principles including encapsulation, class design, file I/O, and
array-based data management. All projects are console-based and use no external
libraries.

| Task | Description |
|------|-------------|
| CGPA Calculator       | Multi-semester GPA/CGPA tracker |
| Banking System        | Accounts, deposits, transfers, history |
| Login & Registration  | File-backed auth with password hashing |
TASK 1
A console-based CGPA Calculator built in C++ using Object-Oriented Programming.
Supports multiple semesters and courses per semester. Each course stores its name,
grade (0.0–4.0 scale), and credit hours. The program computes per-semester GPA and
the overall Cumulative GPA (CGPA), then displays a formatted academic report.
Key Concepts: OOP (Classes, Encapsulation), Arrays, Const Methods, Formatted I/O
TASK 2
A console-based Banking System built in C++ using Object-Oriented Programming.
Features a 4-class architecture (Transaction, Account, Customer, Bank) supporting
customer registration, multi-account management, deposits, withdrawals, and fund
transfers between accounts. Each account maintains a full transaction history
viewable on demand.
Key Concepts: OOP (Classes, Encapsulation), Arrays, Pointers, Menu-driven Interface
TASK 3
A console-based Login and Registration System built in C++ using Object-Oriented
Programming. User credentials are persisted to a flat file (users.txt) with passwords
secured via djb2 hashing — never stored in plain text. Includes input validation
(username format, password strength), duplicate username detection, password
confirmation on registration, and masked password input at runtime.

Key Concepts: OOP (Classes, Static Methods), File I/O, Password Hashing,
Input Validation, Cross-platform Hidden Input
